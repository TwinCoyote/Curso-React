# Cursos-React

Este repositorio contiene todos los ejercicios, proyectos y prácticas que realizo mientras aprendo React y refuerzo conceptos fundamentales de JavaScript, TypeScript y desarrollo frontend.

La carpeta está organizada por módulos, donde cada uno incluye ejemplos prácticos, retos y pequeñas aplicaciones creadas durante el proceso de aprendizaje.

El objetivo del repositorio es documentar mi progreso, practicar buenas prácticas y mantener un historial de todo lo que voy aprendiendo en el camino.
